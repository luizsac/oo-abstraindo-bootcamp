## Sobre

Repositório do projeto "Abstraindo um Bootcamp Usando Orientação a Objetos em Java" da plataforma DIO.

Trata-se de um projeto para exercitar os conceitos de Orientação a Objetos. É uma aplicação 
[Gradle](https://gradle.org/) que utiliza o [Lombok](https://projectlombok.org/) para a geração de código boilerplate e 
[JUnit 5](https://junit.org/junit5/) para criaçãod e testes unitários.
